This datapack makes the Nether coordinates 1:1 with the Overworld. Designed to encourage using the Overworld to travel!


## How do I change the scale?
*Just change the "coordinate_scale" value in [the_nether.json](https://github.com/tersordo/1-to-1-nether/blob/main/data/minecraft/dimension_type/the_nether.json)*
